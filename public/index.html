üë• Players Endpoints

GET /api/players
Haal alle spelers op met paginatie en filters
Query params: ?limit=10&offset=0&sortBy=name&order=ASC&position=Keeper&active=true
GET /api/players/search
Zoek spelers op naam of positie
Query params: ?name=John&position=Aanvaller
GET /api/players/:id
Haal een specifieke speler op
GET /api/players/:id/stats
Haal alle statistieken van een speler op (goals, assists, kaarten, etc.)
GET /api/players/:id/injuries
Haal alle blessures van een speler op
POST /api/players
Maak een nieuwe speler aan
Body: { "name": "Jan Janssen", "jersey_number": 10, "position": "Aanvaller", "birth_date": "1995-05-15", "active": true }
PUT /api/players/:id
Update een bestaande speler
DELETE /api/players/:id
Verwijder een speler

‚öΩ Matches Endpoints

GET /api/matches
Haal alle wedstrijden op met paginatie
Query params: ?limit=10&offset=0&sortBy=match_date&order=DESC&status=completed
GET /api/matches/search
Zoek wedstrijden op tegenstander, datum of status
Query params: ?opponent=FC%20United&date=2024-01-15&status=completed
GET /api/matches/upcoming
Haal alle aankomende wedstrijden op
GET /api/matches/completed
Haal alle afgeronde wedstrijden op
GET /api/matches/:id
Haal een specifieke wedstrijd op
POST /api/matches
Maak een nieuwe wedstrijd aan
Body: { "match_date": "2024-01-20", "match_time": "20:00", "opponent": "FC United", "location": "Schepdaal", "home_away": "home", "status": "scheduled" }
PUT /api/matches/:id
Update een wedstrijd (inclusief score)
DELETE /api/matches/:id
Verwijder een wedstrijd

üìä Match Statistics Endpoints

GET /api/matches/:id/stats

POST /api/matches/:id/stats

PUT /api/stats/:id
Update wedstrijdstatistieken
DELETE /api/stats/:id
Verwijder wedstrijdstatistieken

üè• Injuries Endpoints

GET /api/injuries
Haal alle blessures op
GET /api/injuries/active
Haal alleen actieve blessures op
POST /api/injuries
Registreer een nieuwe blessure
Body: { "player_id": 1, "injury_type": "Hamstring", "description": "Overbelasting", "injury_date": "2024-01-10", "expected_return_date": "2024-02-01", "status": "active" }
PUT /api/injuries/:id

DELETE /api/injuries/:id
Verwijder een blessure

üîí Validatie Regels

  • Namen: Alleen letters en spaties
  • Rugnummers: 1-99, uniek per speler
  • Posities: Keeper, Verdediger, Middenvelder, Aanvaller
  • Scores: Moet >= 0 zijn
  • Gespeelde minuten: 0-60 (zaalvoetbal)
  • Goals/Assists/Kaarten: Moet >= 0 zijn
  • Datums: expected_return_date moet na injury_date
